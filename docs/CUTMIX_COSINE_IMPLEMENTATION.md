## üêõ Troubleshooting

### Common Issues

**"MixUp disabled" message**:
- Check that `mixup_alpha > 0` or `cutmix_alpha > 0` in config

**Scheduler not found**:
- Verify `SCHEDULER_TYPE = 'cosine_annealing'` in config file

**Import errors**:
